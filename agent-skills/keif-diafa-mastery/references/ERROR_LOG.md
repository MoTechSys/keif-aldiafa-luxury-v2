## 📂 سجل الأخطاء والتحليلات (Log & Analysis)

### 1. مشكلة حظر محركات البحث (SEO Block)
- **التاريخ:** 2026-03-11
- **المشكلة:** ملف `robots.txt` كان يحظر مجلد `/_next/` مما يمنع Google من قراءة ملفات CSS و JS.
- **السبب الجذري:** إعدادات افتراضية صارمة جداً في `src/app/robots.ts` لم تراعِ متطلبات Next.js الحديثة.
- **الحل المعماري:** إزالة `/_next/` من قائمة الحظر (Disallow) وتحديث القواعد لتشمل الـ Sitemap.
- **تدقيق الفوضى:** قد يؤدي هذا الحظر إلى فشل الـ LCP (Largest Contentful Paint) لأن المتصفح لا يستطيع تحميل الـ CSS.
- **الوقاية:** مراجعة ملف `robots.txt` دائماً عند أي تحديث لـ Next.js واستخدام أدوات فحص الـ SEO التلقائية.

### 2. مشكلة الخطوط غير المتناسقة (Font Inconsistency)
- **التاريخ:** 2026-03-11
- **المشكلة:** العناوين كانت تظهر بخطوط افتراضية غير فاخرة.
- **السبب الجذري:** عدم دمج خط `Amiri` بشكل صحيح في `layout.tsx` وعدم تعريفه كـ CSS Variable عالمي.
- **الحل المعماري:** استيراد خط `Amiri` من `next/font/google` وتطبيقه عبر `CSS Variables` في `globals.css` مع تفعيل الـ `swap` لضمان سرعة العرض.
- **تدقيق الفوضى:** تأخر تحميل الخطوط قد يسبب CLS (Cumulative Layout Shift).
- **الوقاية:** استخدام `BRAND_GUIDE.md` كمرجع أساسي وتفعيل الـ `preload` للخطوط الأساسية.
